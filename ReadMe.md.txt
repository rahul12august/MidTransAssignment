# Front end Automation by XSEED QA

# Following software needs to be install to execute Selenium Scripts
* Download and install latest Chrome Browser
* Download Chrome driver so that Selenium can interact with Chrome Browser by using following command in case of Ubuntu OS:
	sudo apt-get install unzip
	wget -N http://chromedriver.storage.googleapis.com/2.38/chromedriver_linux64.zip
	unzip chromedriver_linux64.zip
	chmod +x chromedriver
	sudo mv -f chromedriver /usr/local/share/chromedriver
	sudo ln -s /usr/local/share/chromedriver /usr/local/bin/chromedriver
	sudo ln -s /usr/local/share/chromedriver /usr/bin/chromedriver
* If you are using MAC OS or Windows OS, no need to install chrome driver. Just download Chrome Driver for respective OS and keep it in project     workspace.
* install IDK and set JAVA_HOME path.
	
* install git 

* install Maven and set maven path
	
* Go to any project workspace and run command : mvn install
